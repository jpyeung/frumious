Word play

Finds words matching a list of conditions, one per line. The most common words are returned first.

Regular expressions

You can use regular expressions such as:

..sw.. - 6-letter words with 'sw' in the given position

.*(...)\1.* - Words with a 3-letter repetition

"!expression" inverts an expression.
"\P" matches an element from the periodic table.
"\S" matches a US state abbreviation.

Operations

You can also list operations such as:

anagram: dirtyroom

add 3: pinky - Add three letters to get a word

The full list of commands is:

anagram: <letters>
Rearranging the given letters
subanagram: <letters>
Rearranging at most the given letters
superanagram: <letters>
Rearranging at least the given letters
transdelete [n]: <letters>
Rearranging all but N of the given letters (N=1 if omitted)
transadd [n]: <letters>
Rearranging all of the given letters plus N wildcards (N=1 if omitted)
bank: <letters>
Using the same set of unique letters as the given string
subbank: <letters>
Using at most the same set of unique letters as the given string
superbank: <letters>
Using at least the same set of unique letters as the given string
substring: <letters>
Included in the given string
delete [n]: <letters>
Achievable by deleting N letters from the given string (N=any if omitted)
add [n]: <letters>
Achievable by inserting N letters into the given string (N=any if omitted)
change [n]: <letters>
Achievable by exactly N letter changes from the given string (N=1 if omitted)

Cross-filtering

This is a mechanism for finding related pairs of words. For example, to find words which are transdeletions of another 8-letter word:

.{8}
=>
transdelete: %

The "=>" operator divides the list of filters into two stages. The first stage is a regular expression that finds 8-letter words. For each of the top results from the first stage, the second stage is executed with the "%" symbol replaced by the word in question. The resulting pairs of words are shown.

Up to 1000 matches from the first stage are considered, and 20 matches from the second stage.
