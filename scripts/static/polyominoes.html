Polyominoes

Assembles a set of polyominoes into a given shape. The input is a mode, followed by a board shape, then the pieces, separated by empty lines. The mode is one of:

translation - only translation allowed
rotation - translation and rotation allowed
reflection - translation, rotation, and reflection allowed

If there are not enough pieces to fill the board, results with empty space will be returned. If the piece area is greater than the board area, pieces will be allowed to overlap to fill the board.
